# 💰 Finance Dashboard

Aplicación de finanzas personales con diseño dark elegante. Registra ingresos, gastos, inversiones y ahorros con visualización en dashboard.

## Características

- **Dashboard** con KPIs, gráficos de tendencia y donut de categorías
- **Movimientos** con filtros por tipo y edición inline
- **Inversiones** con distribución de portfolio y evolución
- Persistencia local con `localStorage`
- Diseño dark mode con animaciones suaves

## Desarrollo local

```bash
npm install
npm run dev
```

## Despliegue en Vercel

1. Sube este repo a GitHub
2. Ve a [vercel.com](https://vercel.com) y conecta el repo
3. Vercel detectará Vite automáticamente — haz click en **Deploy**

## Despliegue en GitHub Pages

```bash
npm run build
# Sube la carpeta dist/ a la rama gh-pages
```

## Tech Stack

- React 18
- Vite 6
- CSS-in-JS (inline styles)
- Google Fonts (DM Sans + JetBrains Mono)
